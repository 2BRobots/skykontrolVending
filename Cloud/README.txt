1. Import the "skykontrol_DB" file to your server and use it to create the database.
2. Edit "credentials.php" with your users and passwords for your server. 
3. Copy skykontrol folder to your server.
4. Enjoy and share improvements.

Also you will need to add PHP files in the "scripts" folder to a cron job to run at desired times.

Note: the console is under development!
